### Install Chrome

For example, exhibiting projects with a web browser

Install Chrome [reference](https://raspberrypi.stackexchange.com/questions/374/how-do-i-install-google-chrome)

```bash
# update
sudo apt-get update
# install browser
sudo apt-get install chromium-browser
```
